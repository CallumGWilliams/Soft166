About the project:

This project is a HTML and JavaScript language based password strength checker that has been developed by myself, Callum Gorman-Williams as a part of my module assessment for Bsc Computing. The programme contains a suitably designed algorithm to calculate the strength of the password entered by the user. Furthermore, this website has been designed to incorporate programmable Phillips-Hue smart lights which display the strength of the password in terms of colours displayed on the lights themselves.

The algorithm is designed to test the length of the password entered, as well as whether or not it contains upper case characters, numbers and symbols.
